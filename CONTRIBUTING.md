# Contributing

## Improve this boilerplate

Found something wrong, outdated, or missing? Open a PR directly — no issue needed for small fixes.

## PR checklist

- [ ] Changes tested against the current version of the agent
- [ ] Examples verified against the current `openai/codex` docs or CLI help
- [ ] README updated if files/structure changed
- [ ] No fake commands or raw placeholder markers left in template files

## What makes a good contribution

- **Accurate** — tested against current agent version, not guessed
- **Practical** — real working files, not just docs
- **Concise** — comments explain the why, not the obvious

## Adding a new agent

Wrong repo? Head to [agent-anatomy](https://github.com/agent-anatomy/agent-anatomy) to propose a new agent boilerplate.

## Questions

Open an issue.
